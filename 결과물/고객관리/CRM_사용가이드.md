# 킹옥션 CRM 사용 가이드 📒

이사님, 새롭게 제작된 고객 관리 시스템을 200% 활용하는 방법입니다!

## 1. 시스템 접속 방법
- `결과물/고객관리/킹옥션_CRM_시스템.html` 파일을 더블 클릭하면 브라우저(크롬 등)에서 바로 열립니다.
- 별도의 로그인이 필요 없으며, 입력한 데이터는 이사님의 컴퓨터에만 안전하게 저장됩니다.

## 2. 주요 기능 활용법
- **고객 등록**: 우측 상단 **[고객 추가]** 버튼을 눌러 새로운 연락처를 저장하세요.
- **퍼널 단계 변경**: 고객 정보의 **[수정(✏️)]** 아이콘을 눌러 '상담 중'이나 '수강 완료'로 상태를 바꿀 수 있습니다.
- **데이터 저장**: 따로 저장 버튼을 누를 필요 없이 입력 즉시 자동 저장됩니다.
- **AI 스마트 입력 (강력 추천!)**: 
    - 화면 상단의 **[✨ AI 스마트 입력]** 버튼을 누르세요.
    - 카톡 대화 내용이나 문자 메시지를 통째로 복사해서 붙여넣으세요.
    - **[분석하기]**를 누르면 제가 이름, 번호, 단계를 자동으로 찾아냅니다.
    - 확인 후 **[CRM에 등록]**만 누르면 끝! (직접 타이핑할 필요가 없습니다.)
- **엑셀 출력**: 상담 리스트를 엑셀로 보고 싶을 때 **[내보내기]** 버튼을 누르면 CSV 파일로 다운로드됩니다.

## 3. 이사님을 위한 팁 💡
- **메모장 활용**: '특징' 칸에 어제 대화 나눈 사소한 내용(예: "경매 공부 1년 차", "소액 투자 관심")을 적어두시면 나중에 다시 연락할 때 큰 힘이 됩니다.
- **정기적 확인**: 매주 한 번씩 '상담 중'인 고객분들에게 안부 인사를 보내 수강 전환을 유도해 보세요.

이제 더 이상 연락처를 메모지에 적지 마시고, 킹옥션 CRM으로 스마트하게 관리하세요! 🚀
